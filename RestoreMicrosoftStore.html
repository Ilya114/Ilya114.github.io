Restore Microsoft Store

Sometimes UUPDump not downloading some files and Microsoft Store can lacks in your Windows build. Write this command in PowerShell to restore Microsoft Store: wsreset -i. You should start seeing installation progress in the notification center. If this not helped, you should download and install Microsoft Store manually:

  • Get access to Microsoft Store .appx from e.g AdGuard Store (you need to write https://apps.microsoft.com/store/detail/microsoft-store/9WZDNCRFJBMP?hl=en-us&gl=us link to get access to .appx.)
  • Download Microsoft Store .appx (example: Microsoft.WindowsStore_22212.1401.5.0_neutral_~_8wekyb3d8bbwe.msixbundle) and Microsoft.UI.Xaml .appx (example: Microsoft.UI.Xaml.2.4_2.42007.9001.0_arm64__8wekyb3d8bbwe.appx) from the page opened above
  • Run Windows Powershell with Admininistrator rights, go to .appx path and install Microsoft.UI.Xaml first: Add-AppxPackage -Path .\Microsoft.UI.Xaml.2.7_7.2208.15002.0_arm64__8wekyb3d8bbwe.Appx, then install Microsoft Store: Add-AppxPackage -Path .\Microsoft.WindowsStore_22210.1401.6.0_neutral___8wekyb3d8bbwe.Msixbundle
