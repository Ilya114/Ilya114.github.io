Update drivers instructions
Start TWRP recovery through the PC with the command: fastboot boot `recovery.img`
If you already have TWRP installed, just hold the power and vol+ buttons at startup

Execute script msc.sh: adb shell msc.sh

Assign letters to partitions
Once the X3 Pro is detected as a disk launch diskpart
Use list volume to find Windows and ESP partitions, it's the ones named "WINVAYU" and "ESPVAYU"
# select volume
# assign letter=x
# select volume
# assign letter=y
# exit

Install drivers
Open cmd as administrator and go to driverupdater location
Replace `vayudriversfolder` with the location of the drivers folder
.\driverupdater.exe -d `vayudriversfolder`\definitions\Desktop\ARM64\Internal\vayu.txt -r `vayudriversfolder` -p X:
