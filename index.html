Windows on POCO X3 Pro - this is a project to launch Windows ARM on Xiaomi POCO X3 Pro phone. You can try the full Windows experience¹ on your POCO X3 Pro today.

Some instructions were taken and adapted from the GitHub project repository. All programs belong to their authors.

⚠️ Warning! This is experimental software. We are not responsible for the bricks you received while trying to install Windows on your POCO X3 Pro.
